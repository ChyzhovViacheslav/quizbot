Перед использованием бота нужно установить все зависимости и запустить сервер локально
Сам бот - @quiz_telegraf_test_bot (https://t.me/quiz_telegraf_test_bot)
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
